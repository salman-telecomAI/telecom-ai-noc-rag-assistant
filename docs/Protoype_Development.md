Develop the first working version of the Telecom AI NOC RAG Assistant.

## Components

- FastAPI Backend
- REST API
- Document Loader
- RAG Pipeline
- Vector Database
- Azure OpenAI Integration
- Chat Endpoint

User
↓
FastAPI
↓
RAG Engine
↓
Vector Database
↓
Azure OpenAI
↓
Response

- Backend API

- Local Testing
- Working Prototype

A functional AI assistant capable of answering telecom questions using Retrieval-Augmented Generation (RAG).
