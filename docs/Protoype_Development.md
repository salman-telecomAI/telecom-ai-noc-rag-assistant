# Prototype Development

## Purpose

This document describes the development process of the Telecom AI NOC RAG Assistant from the initial concept to the final working prototype.

The project was developed incrementally, with each phase adding new functionality while maintaining a simple and modular architecture.

---

# Project Vision

The objective was to build an AI-powered Telecom Network Operations Center (NOC) Assistant capable of answering telecom engineering questions using Retrieval-Augmented Generation (RAG).

The solution was designed to demonstrate practical AI engineering skills while remaining simple enough for learning, demonstration and technical interviews.

---

# Development Approach

The project followed an iterative development approach.

Each phase introduced a small set of features that were tested before progressing to the next phase.

This reduced complexity and ensured that every component functioned correctly before adding new capabilities.

Several challenges were encountered during development.

- Managing Azure OpenAI configuration
