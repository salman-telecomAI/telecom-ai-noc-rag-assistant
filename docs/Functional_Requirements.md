# Functional Requirements

## FR-01
The system shall accept natural language questions from users.

## FR-02
The system shall retrieve relevant telecom documents using Retrieval-Augmented Generation (RAG).

## FR-03
The system shall generate accurate responses using Azure OpenAI.

## FR-04
The system shall cite the source documents used to generate answers.

## FR-05
The system shall support searching across manuals, SOPs, RCA documents, alarm references, and network documentation.

## FR-06
The system shall maintain conversation context during user interactions.

## FR-07
The system shall allow future integration with Azure AI Search and enterprise authentication.
