# Metadata Design

## Objective

Define metadata fields for telecom documents to improve semantic retrieval and document management.

## Standard Metadata

- Document Name
- Vendor
- Technology
- Document Type
- Alarm Type
- Version
- Source
- Page Number
- Author
- Date

## Example

Document Name: Huawei DWDM Manual

Vendor: Huawei

Technology: DWDM

Document Type: Vendor Manual

Alarm Type: LOS

Version: V3.2

Page: 125

## Benefits

- Efficient filtering
- Improved retrieval accuracy
- Better document organization
- Easier maintenance

## Expected Outcome

Every document chunk contains descriptive metadata that improves semantic search, filtering and retrieval quality.
