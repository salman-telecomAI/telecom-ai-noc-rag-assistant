# Document Types

## Purpose

Organize enterprise telecom documents into logical categories for efficient Retrieval-Augmented Generation (RAG).

## Categories

### Vendor Manuals
- Huawei DWDM
- Nokia Optical
- Cisco IP/MPLS

### Standard Operating Procedures (SOP)
- Fault Handling
- Planned Maintenance
- Service Restoration

### Root Cause Analysis (RCA)
- Incident Reports
- Post-Incident Reviews

### Alarm Guides
- LOS
- LOF
- AIS
- BER
- OSNR

### Network Documentation
- Topology
- Design Documents
- Fiber Routes
- IP Addressing

### Operational Runbooks
- Daily Operations
- Escalation Procedures
- Emergency Procedures

## Expected Outcome

Structured document organization improves search relevance, simplifies maintenance, and increases AI response accuracy.
